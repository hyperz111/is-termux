# is-termux

Check if you run in [Termux](https://termux.dev/en/) environment.

## Install

```sh
$ npm install is-termux
```

## Usage

```js
import isTermux from 'is-termux';

isTermux(); // => "true" if you in Termux
```

## API

### `isTermux()`

Return: `boolean`

Check if you run in Termux environment.
